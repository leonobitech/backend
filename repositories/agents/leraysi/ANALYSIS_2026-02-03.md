## Problemas Críticos

### 1. Sin Error Handling
**Impacto:** Bot muere silencioso, clienta sin respuesta
**Ubicación:** Todo el sistema
**Solución:** Error workflow global + try-catch en Code nodes

### 2. Código Duplicado
**Impacto:** Bugs, difícil mantenimiento
**Ubicación:**
- `Chat History Filter` y `Chat History Filter1` (100% idénticos)
- `ComposeProfile` y `LoadProfileAndStateImage` (80% idénticos)
**Solución:** Consolidar en nodos únicos o sub-workflows

### 3. Nodos Muertos/Desconectados
**Impacto:** Confusión, uso de memoria
**Nodos:**
- `Qdrant Vector Store1` - no conectado
- `Simple Memory` - no conectado
- `odoo_schedule_meeting` - tool definida pero no adjunta al Master Agent
- `odoo_send_email` - tool definida pero no adjunta al Master Agent
**Solución:** Eliminar o conectar correctamente

### 4. Redis Sin TTL
**Impacto:** Memory leak potencial
**Ubicación:** Buffer de mensajes (PushBufferEvent)
**Solución:** Agregar TTL de 1 hora a las keys

### 5. Cancelar Turno Sin Validación
**Impacto:** Inconsistente con Crear/Reprogramar
**Ubicación:** Sub-workflow `Leraysi - Cancelar Turno`
**Solución:** Agregar Code node de validación como los otros

### 6. Tool cancelar_turno Conectada pero No Documentada
**Impacto:** LLM podría llamarla incorrectamente
**Ubicación:** Agente Calendario
**Solución:** Documentar en system prompt o desconectar

### 7. Sin Detección de Colisión de Horarios
**Impacto:** Posible doble-booking en mismo slot
**Ubicación:** AnalizarDisponibilidad
**Solución:** Agregar verificación por hora, no solo por día

### 8. Output Main Hace Demasiado
**Impacto:** 200+ líneas, frágil, difícil debug
**Ubicación:** Sales_Agent_By_WhatsApp
**Solución:** Dividir en 3 nodos: JSONParse, StateMerge, OutputFormat

### 9. Valores Hardcodeados
**Impacto:** Inflexible para otros clientes
**Ejemplos:**
- Timezone `-03:00` hardcodeado
- Odoo IDs (country_id: 10, state_id: 553)
- URLs internas (piper_tts:5000, chatwoot:3000)
- Capacidad diaria (6 turnos, 2 pesados, 1 muy pesado)
- Porcentaje seña (30%)
**Solución:** Extraer a workflow variables o config

## Lo Que Está Bien ✓

### Buffer Redis 8 Segundos
Resuelve el problema de mensajes fragmentados de WhatsApp. Patrón sólido.

### Two-Tier AI
- Welcome Agent (GPT-4.1-mini): saludo inicial, bajo costo
- Master Agent (GPT-4o): ventas complejas, herramientas
Buena separación de responsabilidades.

### Calendario Determinístico
Pre-calcula toda la disponibilidad antes del LLM. El agente solo ejecuta, no decide. Patrón robusto.

### Dual Database
- Baserow: estado rápido, lectura/escritura durante conversación
- Odoo: CRM formal, historial permanente
Buena separación de concerns.

### Patrón MCP Adapter
HTTP Request + Code transform funciona aunque no sea MCP nativo.
