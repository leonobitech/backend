# Baserow Schema - Estilos Leraysi

Tablas de Baserow para el agente de WhatsApp del salón de belleza.

## Tablas

| ID | Nombre | Descripción |
|----|--------|-------------|
| 850 | ServiciosLeraysi | Catálogo de 15 servicios del salón |
| 851 | LeadsLeraysi | Leads/contactos desde WhatsApp |
| 852 | TurnosLeraysi | Turnos agendados |
