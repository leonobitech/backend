📋 Cómo conectar con Claude Desktop

Paso 1: Copia esta URL para tu conector personalizado

Paso 2: Abre Claude Desktop y ve a:

  • Menú → Ajustes → Funciones para desarrolladores
  • Haz clic en "Agregar conector personalizado"

Completa el formulario así:

  • 📝 Nombre: Odoo MCP (o el que prefieras)
  • 🔗 URL del servidor MCP remoto: Pega la URL copiada arriba
  • 🔐 OAuth Client ID: Déjalo vacío
  • 🔑 Secreto del cliente OAuth: Déjalo vacío

ℹ️ No necesitas Client ID ni Secret porque este conector usa OAuth 2.1 con PKCE (más seguro para apps de escritorio)

Paso 3: Haz clic en "Agregar" y listo! 🎉
